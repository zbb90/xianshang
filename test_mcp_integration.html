📋 MCP Server工具列表

当MCP Server成功连接后，Cursor将获得以下高德地图工具能力：

🔍 地点搜索

maps_search - 搜索指定关键词的地点信息

🌦️ 天气查询

maps_weather - 查询指定城市的天气信息

🗺️ 地理编码

maps_geocode - 地址转换为经纬度坐标

📍 逆地理编码

maps_regeocode - 经纬度转换为详细地址

🛣️ 路径规划

maps_route - 计算两点间的最优路径

📊 距离测量

maps_distance - 计算多点间的距离

💬 使用示例

配置完成后，你可以在Cursor对话中直接使用这些命令：

// 在Cursor对话框中输入： "帮我搜索北京的星巴克门店" "查询上海今天的天气" "计算从天安门到故宫的路线" "将'北京市朝阳区三里屯'转换为坐标"
🚀 快速开始

现在你可以：

  1. 按 Ctrl/Cmd + L 打开Cursor对话框
  2. 选择 "Agent" 模式确保可以使用工具
  3. 直接用自然语言描述你的地图需求
  4. MCP Server会自动调用相应的高德API
